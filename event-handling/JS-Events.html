Bubbling Javascript Events

Was sind Events und welche Typen gibt es?

  • Aktionen, auf die ich als Entwickler reagieren kann. Und zwar so, wie ich als Entwickler es vorgesehen habe!
  • Werden technisch als "Messages" auf einen Stapel gelegt und FIFO abgearbeitet
  • Events werden seriell abgearbeitet. Solange ein Event abgearbeitet wird, geht nix anderes -> blocking
  • Unschöne Effekte für den User --> "Ein Script auf dieser Seite reagiert nicht mehr" :(
  • Events gibt es auch in anderen Sprachen, nicht nur JS - z.B.: C
  • In manchen Situationen benötigt man kein Javascript! -> zum Beispiel onMouseOver -> CSS :hover
  • Es gibt viele verschiedene :-) Die "gebräuchlichsten":
    • Klick-Event (onclick)
    • Hover-Event (onmouseover)
    • Input-Event (oninput)
    • Change-Event (onchange)
    • Submit-Event (onsubmit)

Vermehrung und Standardverhalten

  • Vermehrung = Propagation
    • Was ist Bubbling
      • Von unten nach oben -> vom innersten zum äußersten Element
      • let clickFn = function(e){ console.log("click"); }
        target.addEventListener('click',clickFn);
    • Was ist Capture
      • Von oben nach unten -> vom äußersten zum innersten Element
      • let clickFn = function(e){ console.log("click"); }
        target.addEventListener('click',clickFn,true);
    • Ich kann es stoppen?! mit event.stopPropagation();
  • Standardverhalten
    • Ist definiert (z.B. Klick oder Form-Submit)
    • Kann unterbrochen werden, mit event.preventDefault(); z.B. für Validierung bei Formularen
    • Dann muss sich der Entwickler aber auch um die Korrektheit kümmern.
